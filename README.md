# timothynoe.com

Personal academic homepage and CV for Timothy Noe, served via GitHub Pages.

## Structure

- `index.html` — the homepage / CV (hand-built, no build tooling).
- `styles.css` — styles for the homepage.
- `CNAME` — custom domain (`timothynoe.com`).
- `assets/` — put your CV PDF (`CV.pdf`) and headshot (`headshot.jpg`) here.
- `index4.9.html` + `style.css` — *The Present Testament* project (now a sub-page,
  linked from the homepage footer).

## Editing the CV

Open `index.html` and fill in the sections marked with `TODO`. Each section
(About, Research, Publications, Education & Experience, Teaching & Service,
Contact) is plain HTML — duplicate a list item to add an entry.

## Publishing

GitHub Pages serves this repo at `https://timdnoe.github.io/` and, once DNS is
configured, at `https://timothynoe.com/`. The live site builds from the
repository's default branch.
